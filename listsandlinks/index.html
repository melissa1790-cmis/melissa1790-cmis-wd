How to Create a New Webpage on the Production Server:

  1. Create a new file named index.html in gedit
  2. Add basic html language and save the index.html file
  3. Create a new file named main.css in gedit
  4. Add basic css content and save the main.css file
  5. Git add index.html and main.css
  6. Git commit
  7. Git push
  8. Open sftp and log in
  9. cd public_html
  10. put index.html and main.css
